## API de la librairie liste-c

createLC: Sert a creer une liste de type "ListeC"  
sizeLC: Renvoie la taille de la liste  
addLC: Ajoute une valeur a la fin de la liste  
getLC: Donne la valeur attribué a une position dans la liste  
replaceLC: Remplace la valeur a une position donné  
insertLC: Insert une valeur a une position donné  
removeLC: Suprime la position donné  
deleteLC: Suprime la liste pour liberer la memoire
